# BooklyX Super Admin Dashboard

A modern web-based administration dashboard for the BooklyX platform.

The dashboard enables platform administrators to manage service providers, approve applications, monitor analytics, review financial data, and oversee the entire booking ecosystem from one centralized interface.

├── API/              # API services and Axios requests

## 🔐 Authentication

The dashboard uses JWT Authentication.

Protected routes prevent unauthorized users from accessing admin pages.

Main features include:

- JWT Authentication
- Analytics APIs
- Branch Management APIs
- Service Management APIs
- Financial APIs
- Plans APIs
